# 福岡・近隣 おすすめかき氷店ガイド

福岡市内、糸島、福岡県南部、北九州、佐賀のおすすめかき氷店12店をまとめた、PC・スマートフォン・印刷対応のWebガイドです。

## ファイル構成

- `index.html`：メインページ
- `fukuoka_kakigori_guide.html`：共有しやすいファイル名のメインページ（`index.html`と同内容）
- `style.css`：デザイン、レスポンシブ、印刷レイアウト
- `script.js`：検索、地域フィルター、お気に入り、共有、ページトップ機能
- `data/shops.js`：店舗データ
- `assets/images/kakigori-hero.png`：このサイト用に生成したヒーロー画像
- `assets/images/shop-*.webp`：各店舗のおすすめを表現したオリジナルイメージ

## 起動方法

フォルダ一式を同じ構成のまま保存し、`index.html` または `fukuoka_kakigori_guide.html` をダブルクリックしてブラウザで開きます。外部ライブラリやビルド作業は不要です。

ローカルサーバーで確認する場合は、このフォルダで次を実行します。

```powershell
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## 店舗情報の追加・修正

`data/shops.js` の `window.SHOPS` 配列を編集します。既存店舗と同じ項目を持つオブジェクトを追加すると、カードと検索へ自動反映されます。

- `id`：英数字の一意なID
- `name`：店名
- `area`：`福岡市内`、`糸島`、`福岡県南部`、`北九州`、`佐賀`のいずれか
- `city`：市区町村・所在地
- `description`、`recommendation`、`caution`：紹介文
- `tags`：検索対象となるタグ配列
- `officialUrl`、`instagramUrl`：確認できた公式リンク。未確認なら空文字
- `mapQuery`：Googleマップで検索する店名と地域
- `rank`：上位5店だけ1〜5を指定
- `featured`：本命店のみ`true`

## 画像の差し替え

ヒーロー画像を差し替える場合は、`assets/images/kakigori-hero.png` を同名・同形式の画像に置き換えます。推奨比率は横3：縦2、推奨サイズは1536×1024px以上です。権利を保有する画像または利用許可のある画像だけを使用してください。

店舗カードは、権利未確認写真の無断転載を避けるため、各店の特徴とおすすめ内容をもとに作成したオリジナルのイメージ画像を使用しています。実際の提供商品とは盛り付けや内容が異なる場合があります。

## PDF保存・印刷

ページ上部の「PDF・印刷」ボタンを押します。印刷画面でプリンターを「PDFに保存」に変更し、A4縦・倍率100％前後で保存してください。検索欄、お気に入りボタン、共有ボタンなどは印刷時に自動で非表示になります。

## OneDriveで共有する方法

フォルダ一式をOneDriveに置き、フォルダ単位で共有します。受け取った人はファイル一式をダウンロードし、`index.html`を開けます。

OneDriveの共有リンクからHTMLを開くと、Webページ表示ではなくプレビューまたはダウンロードになる場合があります。また、HTMLだけを単独共有するとCSS・JavaScript・画像が読み込まれません。URLとしてそのまま閲覧してもらう用途には、次の静的サイト公開サービスがおすすめです。

## 公開方法

### GitHub Pages（おすすめ）

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルを構成のままアップロードします。
3. リポジトリの「Settings」→「Pages」を開きます。
4. 「Deploy from a branch」を選び、`main`ブランチの`/(root)`を指定します。
5. 表示された公開URLを共有します。

### Netlify

Netlifyの管理画面で、このフォルダをドラッグ＆ドロップすると公開できます。更新時は同じサイトへフォルダを再アップロードします。

### Cloudflare Pages

GitHubリポジトリをCloudflare Pagesへ接続し、フレームワークなし・ビルドコマンドなしで公開します。出力ディレクトリにはルートを指定します。

## リンク確認

掲載リンクは2026年7月18日に公式サイト、公式Instagram、運営会社情報を優先して確認しています。季節営業や移転があるため、公開前・お出かけ前に各リンクを開いて再確認してください。確認できない公式サイトのボタンは表示していません。Googleマップは店名と地域名による検索URLです。

## お気に入りについて

ハートで登録した「行きたい店」はブラウザの`localStorage`へ保存されます。別のブラウザや端末には同期されず、ブラウザデータを削除すると消えます。
